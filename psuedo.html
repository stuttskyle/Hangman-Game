Hangman PseudoCode

1. User chooses a letter on the keyboard to guess a letter of the word.

2. User may use a hint if they need one.

3. We determine if the user wins or not.

  • If user picks a correct letter, they continue with no penalty.

  • If user picks the wrong letter, they lose a guess.

  • If user runs out of guesses, then user loses.

4. We then add to their score.

  • If user wins then we add one to their wins.

  • If user loses then we add one to their losses.

5. The user then presses any key to play again.
